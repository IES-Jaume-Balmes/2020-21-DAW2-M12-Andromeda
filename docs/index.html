Andrómeda

  • Andrómeda es una aplicación web que permitirá a los usuarios encontrar su música favorita, añadir y/o borrar canciones a sus listas, crear, borrar y/o modificar listas personalizadas para compartirlas con sus amigos.
  • Andrómeda contendrà una base de datos relacional sql de canciones.
  • Se podrá visualizar la cantidad de reproducciones de cada canción y dispondrá de filtros por género, autor y tendencias.
  • Creación de una App móvil Android y de una landing page de la aplicación.
