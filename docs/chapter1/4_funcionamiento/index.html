Funcionamiento

  • Dos perfiles de usuario, el normal y el administrador.
  • Usuario:
  • Registro: el usuario podrá registrarse.
  • Baja: el usuario podrá darse de baja del servicio.
  • Login: una vez registrado puede loguearse.
  • Logout: un usuario puede desconectarse.
  • El usuario podrá acceder a su panel de control.
  • Red social de Andrómeda embedded.
  • Correo electrónico al completar el registro.
  • Música:
  • Buscar: el usuario puede buscar canciones.
  • Crear: un usuario puede subir música a Andrómeda.
  • Modificar subidas: un usuario puede modificar sus subidas.
  • Modificar y borrar.
  • Listas de música:
  • Crear: el usuario puede crear listas públicas y privadas.
  • Borrar propias: el usuario puede borrar sus listas.
  • Listar propias.
  • Guardar listas de otros usuarios.

  • Añadir a tus listas.
  • Quitar de tus listas.
  • Buscar en listas.
